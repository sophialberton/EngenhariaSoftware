Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It provides structure to a web page, using elements such as <h1>, <p>, and <div>.

HTML Example:

<h1>Welcome to the Web!</h1>
<p>This is a paragraph element that explains content.</p>

Introduction to CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. It allows you to add colors, fonts, and layouts to make your web pages visually appealing.

CSS Example:

body {
  background-color: lightblue;
  color: navy;
}
h1 {
  font-size: 2.5em;
  text-align: center;
}

Introduction to JavaScript

JavaScript is a programming language that makes web pages interactive. You can create alerts, dynamic elements, or even modify the content of the page based on user actions.

JavaScript Example:

<button onclick="greetUser()">Click Me</button>
